Twitter(X) GIF Saver 🐦 🎬
Twitter (X) のツイートURLから、アップロードされたGIF画像（内部的にはMP4形式）や動画を抽出し、ブラウザ上で手軽にダウンロード・変換できるWebツールです。
現在のTwitterの仕様である「1つのツイートに複数の動画やGIFが添付されているケース（最大4つ）」にも完全対応しています。
✨ 機能一覧
 * 動画URL抽出: ツイートのURLから裏側のメディア（MP4）URLを自動取得
 * 複数メディア対応: 1ツイートに複数の動画・GIFが含まれる場合、すべて抽出してリスト化
 * 直感的な操作UI: サムネイルのタップや、動画プレビュー部分の左右スワイプによる快適な切り替え
 * 高画質MP4ダウンロード: CORS制限を回避し、ブラウザから直接MP4ファイルを保存（複数保存時はファイル名に自動で連番を付与）
 * GIF変換ダウンローダー: クライアントサイド（ブラウザ内）でMP4を軽量なGIF画像にエンコードして保存
 * PWA対応: スマホやPCの「ホーム画面に追加」から、ネイティブアプリのようにインストール可能
📖 使い方
 * Twitter (X) アプリやブラウザで、保存したいGIFや動画が含まれるツイートの「共有」ボタンからリンクをコピーします。
 * 本ツールを開き、入力欄にコピーしたURL（例: https://x.com/user/status/123456...）を貼り付けます。
 * 「抽出する」 ボタンを押します。
 * ツイート情報が取得されると、画面下部にプレビューが表示されます。
   * 💡 複数メディアがある場合: プレビューの下にサムネイル一覧が表示されます。サムネイルをタップするか、メインの動画部分を左右にスワイプ（PCの場合は左右の矢印ボタン）することで、保存したい動画を切り替えることができます。
 * 用途に合わせて以下のいずれかのボタンで保存します。
   * 「MP4のまま保存」: 現在表示している動画ファイルをそのまま保存します。画質が良くファイルサイズも軽いため推奨です。
   * 「本物のGIFに変換して保存」: 現在表示している動画をGIF画像（.gif）に変換して保存します。LINEのスタンプ代わりやブログへの貼り付けなど、画像形式が必要な場合に使用します。
🛠 PWA（アプリ化）としての導入・設置方法
本ツールは静的なファイル（HTML/JSON/画像）のみで動作するため、PHPやNode.jsなどのサーバーサイド環境は不要です。GitHub PagesやVercel、レンタルサーバーなどにアップロードするだけですぐに使用できます。
必要なファイル構成:
/
├── index.html       (メインのHTML/JS/CSS)
├── manifest.json    (PWA用の設定ファイル)
└── icon.png         (アプリアイコン画像 1024x1024推奨、または192x192と512x512)

スマホでのインストール方法:
 * iPhone (Safari): サイトを開き、共有メニュー（四角から矢印が上に出ているアイコン）から「ホーム画面に追加」をタップ。
 * Android (Chrome): サイトを開くと表示される「ホーム画面に追加」バナー、またはメニューから「ホーム画面に追加」をタップ。
📦 使用ライブラリ・API
本ツールは以下のオープンソースライブラリおよびパブリックAPIを利用して構築されています。
フロントエンド / ライブラリ
 * Tailwind CSS (CDN): UIのスタイリングとレスポンシブデザイン。
 * Font Awesome (CDN): UI用のアイコン描画。
 * gifshot.js: Yahooが提供するオープンソースのJavaScriptライブラリ。ブラウザ上で動画をGIFにエンコードするために使用。
外部API / サービス
 * vxtwitter (FixTweet): ツイートのOEmbedメタデータ（動画URLなど）を取得するためのパブリックAPIとして利用。
 * CodeTabs Free CORS Proxy: ブラウザから直接動画をダウンロード（Blob取得）する際に発生するCORS（オリジン間リソース共有）エラーを回避するためのプロキシとして利用。
⚠️ 使用上の注意点
 * 非公開（鍵）アカウントの制限
   * APIの仕様上、非公開アカウント（鍵垢）のツイートURLからは動画を抽出できません。
 * GIF変換の負荷について
   * GIFへの変換処理はサーバーではなくお使いの端末（ブラウザ）のメモリとCPUを使用して行われます。
   * 長時間の動画や、スペックの低いスマートフォンで変換を行うと、処理に時間がかかったり、ブラウザがクラッシュ（強制終了）する可能性があります。（※ツール側で自動的に最大幅320px、10fpsに制限して負荷を軽減しています）
   * ※処理中のクラッシュを防ぐため、変換・ダウンロード処理中は他のメディアへの切り替えスワイプ等をロックする仕様になっています。
 * 外部APIの稼働状況への依存
   * 本ツールは有志が提供する無料のメタデータ取得APIとCORSプロキシに依存しています。これらのサービスが一時的にダウンしている場合や、仕様変更があった場合、動画の取得やダウンロードがエラーになることがあります。
 * 著作権および利用規約の遵守
   * 保存した画像や動画の取り扱いは、Twitter(X)の利用規約および各国の著作権法に従ってください。
   * 他者の著作物を無断で再配布・転載することは禁止されています。あくまで個人の私的利用の範囲内でご使用ください。
